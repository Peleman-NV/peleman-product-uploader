This plugin allows Peleman Industries to upload and maintain WooCommerce data on your WordPress installation.

== Description ==

This plugin exposes a number of API endpoints to allow Peleman's ERP to create and update all items necessary for a webshop, like products, variations, images, tags, categories, etc.
This plugin should only be used by Peleman webshops and client webshops.
